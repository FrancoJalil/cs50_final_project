## Usage:

# For scrape

/scrap

# For export

/export

# Min. command-words

/scrap web.com html_attr

where httml_attr = any attr html (h1, h2, span, p, a, etc)

Example:

/scrap https://quotes.toscrape.com/ span

# Filter for class or id

/scrap web.com html_attr class/id="name"

Example:

/scrap https://quotes.toscrape.com/ span class="text"

# Export any output in a .txt file:

/export https://quotes.toscrape.com/ span class="text"

## TO CONSIDER ##

- Maximum filters: 1 (class or id, not both)

- Exports are .txt files (text/plain)
